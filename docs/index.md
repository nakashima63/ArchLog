# ArchLog

システムアーキテクト向けの技術ニュースアグリゲーターです。

## 対象領域

- **AWS / クラウド設計** - ネットワーク、セキュリティ、可用性、コスト、運用
- **アーキテクチャパターン** - DDD、Hexagonal、CQRS、Event Sourcing、マイクロサービス
- **分散システム基礎** - CAP、整合性、キュー/ストリーム、リトライ、冪等、SLO/SLI
- **インフラ / IaC** - Terraform、CDK、CloudFormation、Kubernetes
- **観測性** - ログ / メトリクス / トレース、OpenTelemetry
- **セキュリティ** - ゼロトラスト、IAM設計、脅威モデリング、OWASP
- **データ基盤** - RDB設計、NoSQL選定、キャッシュ、検索、ETL/ELT
- **重要な新機能 / アップデート** - AWS公式、主要OSSリリースノート

## 使い方

左側のナビゲーションから **News** セクションを開くと、収集された最新記事を閲覧できます。
